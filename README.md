# Global Cancer Risk & Patient Analysis

##  Executive Summary

This project analyzes global cancer patient data to uncover key risk factors, demographic trends, and regional patterns. Using data analysis techniques, the study identifies relationships between lifestyle indicators (such as smoking, alcohol consumption, and obesity) and cancer occurrence. The insights generated can support data-driven decision-making in healthcare planning and preventive strategies.

---

##  Business Problem

Cancer remains one of the leading causes of mortality worldwide. Healthcare organizations require actionable insights to:

* Identify high-risk populations
* Understand contributing risk factors
* Allocate resources efficiently
* Develop preventive healthcare strategies

This project addresses these challenges through structured data analysis.

---

##  Dataset Description

* **Source:** Kaggle – Global Cancer Patients Dataset
* **Records:** Multi-year patient-level data
* **Key Features:**

  * Demographics: Age, Gender, Region
  * Risk Factors: Smoking, Alcohol Use, Obesity, Air Pollution
  * Medical Indicators: Genetic Risk, Cancer Type, Year

---

##  Tools & Technologies

* **Python** (Pandas, NumPy)
* **Data Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebook

---

##  Methodology

The analysis was conducted in the following stages:

1. **Data Preparation**

   * Data cleaning and preprocessing
   * Handling missing and inconsistent values
   * Data type standardization

2. **Exploratory Data Analysis (EDA)**

   * Distribution analysis of demographic variables
   * Risk factor comparisons
   * Outlier detection

3. **Statistical & Correlation Analysis**

   * Identifying relationships between risk factors and cancer incidence
   * Correlation heatmaps and trend evaluation

4. **Trend Analysis**

   * Year-wise and region-wise pattern identification
   * Comparative analysis across population segments

---

##  Key Insights

* Smoking and obesity show a strong positive relationship with cancer risk
* Air pollution significantly contributes to higher cancer incidence in certain regions
* Cancer occurrence increases with age, indicating demographic vulnerability
* Regional disparities highlight the need for targeted healthcare interventions
